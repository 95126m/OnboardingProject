* 데이터베이스: MySQL

### 모듈 구성

* 프론트엔드: 사용자 인터페이스, 데이터 시각화, API 연동
* 백엔드: RESTful API, 비즈니스 로직, 데이터베이스 연동
* 데이터베이스: 직원 정보, 급여 정보, 근태 정보, 휴가 정보
