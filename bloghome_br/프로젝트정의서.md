# 급여 관리 시스템 프로젝트 정의서

## 개요

### 프로젝트명

블로그 서비스 시스템 구축

### 목표

* 로그인 서비스 구축 및 관리
* 블로그 게시글 작성및 관리
* 사용자 친화적인 웹 인터페이스 제공

### 사용자 정의

* 비회원 방문자
* 회원 방문자
* 블로그 관리자

## 요구사항

요구사항 정의서 참고

## 시스템 아키텍처

### 아키텍처 개요

* 프론트엔드: JavaScript (React.js)
* 백엔드: Node.js (Express.js)
* 데이터베이스: MySQL

### 모듈 구성

* 프론트엔드: 사용자 인터페이스, 데이터 시각화, API 연동
* 백엔드: RESTful API, 비즈니스 로직, 데이터베이스 연동
* 데이터베이스: 직원 정보, 급여 정보, 근태 정보, 휴가 정보

### 데이터베이스 설계

데이터베이스 설계서 참고

### 인터페이스 설계

프론트엔드-백엔드 인터페이스: RESTful API를 사용한 데이터 통신
백엔드-데이터베이스 인터페이스: Prisma를 사용한 MySQL 연동

## 설계 상세

### 사용자 스토리
블로그 홈페이지를 관리하고 싶다

#### 블로그 메인 페이지

##### 블로그 메인 페이지 편집
* 블로그 관리자로서, 블로그의 메인 페이지 정보들을 편집하고 싶습니다.

##### 블로그 게시글 정렬
* 비회원, 회원 방문자로서, 블로그 글들을 키워드로 정렬하고 싶습니다.

##### 블로그 게시글 검색
* 비회원, 회원 방문자로서, 블로그 글들을 원하는 키워드로 검색하고 싶습니다.
